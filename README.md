# CS2 Config

Download the repository as a ZIP, extract it, and copy its contents directly into your Steam root directory. The repository layout already matches Steam's directory structure.

- Game CFG files: `steamapps/common/Counter-Strike Global Offensive/game/csgo/cfg`
- Userdata files: `userdata/89582913/730`

After copying, set `userdata/89582913/730/local/cfg/cs2_video.txt` to read-only. This configuration keeps shadows rendered at the lowest quality while allowing a longer shadow distance than the in-game setting exposes.
